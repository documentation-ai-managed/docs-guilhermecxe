title: Create a New Project Documentation Space
description: Step-by-step: create and name a new doc space in guilhermecxe.
---

## Guide

<Steps>
  <Step title="Go to Dashboard">
    Log in at `https://guilhermecxe.com` and open your dashboard.
  </Step>
  <Step title="Start New Doc Space">
    Click the **New Doc Space** button.
  </Step>
  <Step title="Enter Details">
    Name your doc space and (optionally) add a description for your project.
  </Step>
  <Step title="Finish">
    Click **Create**. Your new documentation area appears on the dashboard.
  </Step>
</Steps>
