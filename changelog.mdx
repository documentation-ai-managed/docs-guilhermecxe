<Update label="2024-10-20" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added TikTok Ads integration: Connect your TikTok campaigns directly and analyze performance alongside Google and Meta data.
- AI-powered ROAS forecasting: Ask the AI to predict return on ad spend based on historical data and current trends.

## Improvements

- Enhanced chat interface with natural language understanding for complex queries like "Compare CPA across platforms last week."
- Faster data syncing reduced from 5 minutes to under 60 seconds for daily reports.

## Bug Fixes

- Fixed intermittent dashboard loading issues during peak hours.
- Resolved export errors for CSV files with large datasets.

</Update>

<Update label="2024-09-25" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- LinkedIn Ads support: Now unify LinkedIn metrics with your Google, Meta, and TikTok data for cross-platform insights.
- Custom AI prompts: Save and reuse your most common questions like "Where should I cut ad spend?"

## Bug Fixes

- Corrected calculation errors in LTV metrics for CRM integrations.
- Improved error handling for invalid API keys during platform connections.

</Update>

<Update label="2024-08-15" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Real-time AI agent dashboard: Get instant answers on CAC, ROAS, and CPA without manual spreadsheet work.
- Multi-platform unification: Seamlessly connect Google Ads, Meta Ads, and Analytics in one view.

## Breaking Changes

- Updated authentication flow requires re-connecting platforms with new OAuth scopes for enhanced security.

## Improvements

- Unified data pipeline for automatic syncing every 15 minutes.

</Update>

## Upgrade Guide

Follow these steps to ensure you get the latest features:

<Steps>
  <Step title="Check Current Version" icon="package">
    Log in to your BigLink dashboard and navigate to Settings > About to view your installed version.
  </Step>
  <Step title="Update Integrations" icon="refresh-cw">
    Reconnect any platforms affected by breaking changes using the new OAuth flow.
  </Step>
  <Step title="Test New Features" icon="zap">
    Try the latest AI queries in the chat interface, such as "Forecast ROAS for next month."
  </Step>
</Steps>
