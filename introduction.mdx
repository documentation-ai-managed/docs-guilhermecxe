## Overview

BigLink revolutionizes marketing analysis by unifying data from platforms like Google Ads, Meta, TikTok, and more into a single dashboard. You ask natural language questions, and the AI delivers clear, actionable insights in seconds—helping you decide where to scale campaigns or cut spending without manual spreadsheets. Save hours on data crunching and focus on growth.

## Who Should Use BigLink

BigLink suits marketing teams, agencies, and e-commerce managers overwhelmed by scattered data. Use cases include:

- **Daily performance checks**: Track CPA and ROAS across platforms.
- **Budget optimization**: Identify top-performing campaigns instantly.
- **Agency reporting**: Generate client-ready insights without manual work.

## Quick Start

Get started in minutes:

<Steps>

<Step title="Sign Up" icon="user-plus">

  Create a free account at [biglink.app](https://biglink.app). No credit card required for the 14-day trial.

</Step>

<Step title="Connect Data Sources" icon="link">

  Link Google Ads, Meta, TikTok, and others via OAuth. Data syncs automatically.

  <CodeGroup tabs="Google,Meta">

  ```bash
  # OAuth flow handled in dashboard
  Visit dashboard > Integrations > Connect Google
  ```

  ```bash
  # Meta Business Manager connection
  Authorize BigLink in Meta dashboard
  ```

  </CodeGroup>

</Step>

<Step title="Ask Your First Question" icon="message-circle">

  Open the chat and type `{`Qual foi meu CPA hoje?`}`. Review AI insights and export.

</Step>

</Steps>
